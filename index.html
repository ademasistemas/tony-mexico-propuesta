DETALLES DE LA PLATAFORMA

El sistema le permite tener un control total sobre su operación. A continuación, se detallan las funcionalidades clave:

Gestión Centralizada y Flexible

  • Multi-Empresa: Puede configurar una o más empresas.
  • Inventarios Múltiples: Cree diferentes inventarios o depósitos (ej. "Depósito 01", "Depósito Principal") para cada empresa.
  • Base de Personal: Administre un registro de todos sus empleados para asociarles las asignaciones de equipo.

Control Total de Productos (con campo "Folio")

El módulo de productos es el corazón del sistema. Podrá registrar cada artículo con campos como código, nombre, rubro, descripción, talle, color y, tal como necesita, un campo "Folio" para un seguimiento único. Además, puede gestionar todo masivamente:

  • Importación y Exportación con Excel: Cargue o actualice todo su catálogo de productos de forma masiva, ahorrando horas de trabajo manual.

Flujo de Asignaciones y Devoluciones

Este es el proceso exacto para manejar la entrega de equipo a su personal:

  1. Crea una Solicitud de Salida, seleccionando al empleado y los productos a entregar (camisas, pantalones, equipo, etc.).
  2. Una vez lista, presiona "Confirmar y Generar". Esto descuenta los artículos del stock y los asigna formalmente al empleado.
  3. El sistema emite un Certificado en PDF con el detalle de la entrega, listo para que el empleado lo firme como comprobante de responsabilidad.
  4. Para las devoluciones, el proceso es idéntico pero con una Solicitud de Entrada, reingresando el equipo al inventario.
